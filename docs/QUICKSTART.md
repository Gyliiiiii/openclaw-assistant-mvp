# 快速启动指南

## 🚀 立即运行

在项目目录下执行：

```bash
npm start
```

或者以开发模式运行（会打开开发者工具，方便调试）：

```bash
npm run dev
```

## 📱 使用演示

### 1. 启动应用
运行命令后，会出现一个漂亮的桌面窗口，显示一个 AI 助手角色。

### 2. 测试语音输入
- **按住** 麦克风按钮（中间的大圆按钮）
- 说话："今天有什么邮件"
- **松开** 按钮
- 等待识别和响应

### 3. 使用快捷按钮
点击底部的快捷按钮：
- 📧 **邮件** - 查看今天的邮件
- 📅 **日程** - 查看今天的日程安排

### 4. 查看详细数据
当助手回复后，会自动弹出详细数据面板，显示：
- 邮件列表（包含发件人、主题、预览）
- 日程安排（会议时间、地点）
- 待办任务（优先级、截止时间）

## 🎯 测试命令

试试这些语音命令：
- "今天有什么邮件"
- "邮件"
- "今天的日程安排"
- "今天"
- "汇报"

## 🐛 故障排除

### 语音识别不工作
1. 确保浏览器有麦克风权限
2. 检查系统麦克风是否正常
3. 尝试使用快捷按钮代替语音输入

### 窗口无法拖动
- 在角色区域或对话气泡区域拖动
- 避免在按钮上拖动

### 应用无法启动
```bash
# 重新安装依赖
rm -rf node_modules package-lock.json
npm install
npm start
```
